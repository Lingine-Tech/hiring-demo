# 前端开发测试项目

这是一个用于测试前端开发者能力的半完成项目，基于AIRI虚拟角色平台。

## 项目概述

前端开发测试项目是一个半完成的Vue 3应用程序，包含一些已完成的功能和一些需要实现的功能。项目用于评估候选人的前端开发能力。

## 技术栈

- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **样式**: UnoCSS
- **状态管理**: Pinia

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 测试任务

项目包含以下5个待完成的功能：

1. [消息系统界面](/zh-Hans/docs/tasks/chat-interface)
2. [记忆系统配置](/zh-Hans/docs/tasks/memory-config)
3. [多模态输入](/zh-Hans/docs/tasks/multimodal-input)
4. [实时状态显示](/zh-Hans/docs/tasks/status-display)
5. [用户偏好设置](/zh-Hans/docs/tasks/user-settings)

## 评分标准

- **功能实现** (40%): 功能完整性、交互流畅性
- **代码质量** (30%): 代码结构、可维护性
- **UI/UX设计** (20%): 用户界面、用户体验
- **技术运用** (10%): Vue 3特性、状态管理、类型安全
